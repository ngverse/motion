Introduction

Triggers

Many times, we use animations with triggers like :enter, :leave, etc. Therefore, all animations come with predefined triggers that you can use in your component. There are four predefined triggers:

  • :enter
  • :leave
  • :incr
  • :decr

Depending on the animation version, it may include all four triggers or only entry triggers like :enter and :incr, or exit triggers like :leave and :decr.

Configuration

There are multiple levels of configuration:

  • library configuration
  • animation type configuration
  • animation configuration

Library configuration

You can set global configuration for a whole library. For example in order to set configuration for `animatecss` you can use `setAnimateCssDefaults` methods

Animation type configuration

you can also set configuration per animation type. For example if you want that all your fadeIn animation duration to be 3000ms you can use `setFadeInDefaults`

Animation configuration

You can also set configuration per animation. This is pretty straitforward, as you just pass different params to the animation
