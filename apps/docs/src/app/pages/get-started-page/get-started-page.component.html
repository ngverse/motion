@ngverse/motion

Introduction

is an Angular animation library that implements animations from popular CSS animation libraries in an Angular way using @angular/animations . It is customizable and includes relevant triggers for animations.

Triggers

Many times, we use animations with triggers like :enter, :leave, etc. Therefore, all animations come with predefined triggers that you can use in your component. There are four predefined triggers:

  • :enter
  • :leave
  • :incr
  • :decr

Depending on the animation, it may include all four triggers or only entry triggers like :enter and :incr, or exit triggers like :leave and :decr. Each animation will have available trigger options on their page.

Configuration

You can change the animation configuration like increase duration or change easing function on three levels:

  • Library level configuration
  • Animation type level configuration
  • Animation level configuration

Library level configuration

You can set global configuration for a whole library. This change will apply to all the animations from the specific library

Animation type level configuration

you can also set configuration per animation type. This change will apply to this specific animation type

Animation level configuration

You can also set configuration per animation. This change will be local and apply only to this specific animation

The options are described on defaults page per library and on each animation page
